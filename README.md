#Visitor Pass Management System
A streamlined, modern solution for managing visitor access, enhancing security, and automating the check-in process. This system replaces traditional paper logs with a digital workflow involving QR code authentication, OTP verification, and real-time administrative monitoring

🚀 Features

Self-Service Registration: Visitors can register their details via a clean, responsive web interface.

OTP Verification: Ensures data integrity by verifying visitor mobile numbers before granting access.

QR Code Pass Generation: Automatically generates a unique QR code for every approved visitor for quick scanning at entry points.

Admin Dashboard: A centralized hub for administrators to view active visitors, manage check-ins/check-outs, and access historical logs.

Real-time Status Tracking: Monitor the percentage of project completion and live visitor counts.

🛠️ Tech Stack

Frontend: React.js, Tailwind CSS (Soft Slate & Indigo theme)

Backend: Node.js, Express.js

Database: MongoDB

Authentication: JWT (JSON Web Tokens) and OTP-based verification

Deployment: Vercel (Frontend), Render (Backend)
